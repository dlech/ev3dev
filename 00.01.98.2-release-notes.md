Release notes for ev3dev.img v00.01.98.2
========================================

Changes from previous version
-----------------------------

* Fixed mindboards/ev3dev#43 instead of just working around it
* Uses LVM for root file system partition
* Includes 64MB swap partition
* Provides static MAC address for USB Ethernet

Known issues
------------

* Sound and input port I2C will not work if you remove `console=ttyS1,115200` from the kernel boot parameters (https://github.com/mindboards/ev3dev/issues/47)
* UART and I2C sensors are not detected (correctly) if they are plugged in during boot.


Built using
-----------
* Linux Mint 16 Petra
* polystrap 0.1ev3dev5
* proot 3.2.2-1ev3dev2
* create-image script (https://github.com/dlech/ev3dev-rootfs/blob/9c9d57269bdfb721bbee9884d50b61dc7a5c2497/create-image)
